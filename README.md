# Web Analytics

## التقنيات المستخدمة

- Next.js 16
- TypeScript
- Tailwind CSS
- Recharts
- Framer Motion
- Axios

## التشغيل

```bash
npm install
npm run dev
```

افتح [http://localhost:3000](http://localhost:3000) في المتصفح لرؤية النتيجة.
```

## الصفحات المتاحة

- `/` - نظرة عامة على الإحصائيات
- `/customers` - إحصائيات العملاء
- `/providers` - إحصائيات مقدمي الخدمات
- `/services` - إحصائيات الخدمات
- `/requests` - إحصائيات الطلبات
- `/ads` - إحصائيات الإعلانات
- `/categories` - إحصائيات الفئات

## البناء للإنتاج

```bash
npm run build
npm start
```
